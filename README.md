# 🚀 MOIT Project History

MOIT는 목적형 모임 커뮤니티 플랫폼으로, 기술 스택을 발전시키며 여러 버전으로 개발한 프로젝트입니다.

| Version | Tech Stack                                  | 주요 기능                           |
| ------- | ------------------------------------------- | ------------------------------- |
| **v1**  | Spring Framework, JSP, MyBatis, MySQL, Ajax | 회원가입/로그인, 모임 관리, 문의, 후기, 신고, 광고 |
| **v2**  | Spring Boot, Thymeleaf, MyBatis, Oracle, Ajax, api | Spring Boot 기반으로 리팩토링 및 구조 개선   |
| **v3**  | (예정)                                        | 기능 고도화 및 성능 개선                  |
| **v4**  | (예정)                                        | 추가 기능 개발                        |
